## DIRECTORY TRAVERSAL ATTACK PRINCIPLES & STRATEGIES:

✅ **Flag these traversal patterns:**
- 2+ ../ sequences (reduced from 3+), sensitive paths
- Absolute paths to sensitive files: /etc/passwd
- Encoded traversal: ..%2f..%2fetc/passwd
- Double-encoded traversal: %2e%2e%2f%2e%2e%2f (decodes to ../../..)

❌ **Do NOT flag (normal):**
- Single ../, normal paths like ../images/logo.png
- Relative paths in expected file operations

**Threshold: Flag if ANY of:**
1) encoded traversal
2) 2+ ../ sequences  
3) traversal+sensitive file combination

**IMPORTANT: Decode URL encoding before checking! %2e = ., %2f = /**

## CROSS-SITE SCRIPTING ATTACK PRINCIPLES & STRATEGIES:

🚨 **Flag these XSS patterns (in requests OR responses):**
- Script tags: <script>...任何内容...</script>
- Event handlers: onload=, onerror=, onclick= with JavaScript
- JavaScript protocols: javascript:..., data:text/html
- Malicious function names: maliciousFunction(), exploit(), hack()
- Encoded XSS: %3Cscript%3E, &#x3C;script&#x3E;
- **Check both:** URL parameters AND response content

✅ **Normal (not XSS):**
- Text describing scripts (no tags)
- Code examples in articles
- Expected HTML in content fields
- Special chars in user data
- Email addresses like 'john@gmail.com' in contact field

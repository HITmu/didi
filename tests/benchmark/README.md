# tests/benchmark — 基准测试模块

该目录包含流水线性能与准确性基准测试：

- **`__init__.py`** — 基准测试包初始化
- **`data_generator.py`** — 合成测试数据生成器，生成带标注的数据集用于准确性测试
- **`run_benchmarks.py`** — 基准测试编排器，运行所有基准测试并生成合并报告
- **`test_accuracy.py`** — 准确性基准测试，测量每个流水线阶段的精确率、召回率、F1 分数
- **`test_throughput.py`** — 吞吐量基准测试，测量各阶段独立处理速度（请求/秒）
- **`data/`** — 生成的测试数据文件目录
- **`reports/`** — 基准测试报告输出目录
